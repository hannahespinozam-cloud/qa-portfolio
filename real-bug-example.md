# Real Bug Example

**Title:** Contact form allows submission with empty required fields

**Description:**
The contact form allows users to submit without filling in all required fields, which can result in incomplete or invalid data.

**Steps to reproduce:**
1. Go to the contact page
2. Leave required fields empty
3. Click submit

**Expected result:**
Form should prevent submission and display validation errors

**Actual result:**
Form submits successfully even with empty required fields

**Environment:**
Chrome, Desktop

**Severity:**
High

**Priority:**
High
